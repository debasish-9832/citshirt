1. Installation

After download install packet. You do step by step to install website.


Step 1: Upload the Files to Your Server

Now, you need to upload the extracted files and folders to your web server. The easiest way to upload the installation files is via FTP. For detailed information on how to upload files via FTP.

If you want your site to be the main installation on your account and to be accessible through your main domain (i.e. www.mydomain.com), you need to upload the extracted files to your public_html folder.


Step 2: Create a MySQL Database

Now, you need to create a MySQL database and assign a user to it with full permissions. For detailed instructions on how to do that, please follow the steps described in our tutorial on How to Create MySQL Username and Database. Once you create your MySQL Database and User, make sure you write down the database name, database username and password you've just created. You will need those for the installation process.



Step 3: Run the Installation

Open a new browser window and go to the installation page to run the installation script. Depending on where you installed the script, you will find it at either of the following URLs:

http://yourdomain.com

On this page you will see a message, asking you to prepare the necessary information for the installation and check server supported. Since we already have this information, simply press the "Start Install" button.



Step 4: Add Database Information

Enter the details for your newly created MySQL database and press the button.




Step 5: Add administrative username

On the next screen you will have to enter the information about your administrative username. Once you fill in that information, press the "Run the Install" button. Bear in mind, however, that you should specify a real email address. It can be later used in case you forget your password.




Step 6: Your site installed!

Your new site is installed. You can use the Login In button to access your administrative backend and start add products in your new site.
